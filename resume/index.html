Open Source Projects

Argo Swift

Initially created as an experimentation of Swift's type system and improving the JSON parsing experience in Swift, Argo has turned into a more general applicative parser for moving from loosely-typed structures into strongly typed ones.

Runes Swift

A playground of sorts, defining infix operators to be used by other libraries (such as Argo) as well as some default implementations for the Optional and Array types in the standard lib. The test suite has turned into an experimentation with using property-based testing to verify the various mathematic laws required by conforming to specific interfaces.

reflow Haskell

A Haskell command line tool designed to replace fold -s. It intelligently rewraps lines, with knowledge of things like code blocks, quote blocks, PGP signatures, and plain-text email headers. I use it every day to re-format plain text emails to be more readable. You can see the usage in my dotfiles

vim-xcode Vimscript

A Vim plugin for working with Xcode projects. I'm constantly trying to improve my day-to-day development experience, and a lot of that is focused around trying to break free of Xcode's grasp. vim-xcode allows me to build, test, and run apps directly from within Vim (by wrapping xcodebuild), which helps me avoid needing to switch contexts as often.

Curry Swift

A microframework to bring function currying to Swift. Incredibly useful, especially when paired with Argo, where it allows us to curry a type's initializer instead of requiring a curried constructor to be defined manually. The source for Curry is entirely generated by a script.

Swish Swift

An attempt at building a protocol-oriented networking stack for iOS. The goal here was to build a networking lib that was easily testable and composable. We also wanted to build something that worked nicely with Argo, and returned Result types instead of using Swift's error handling.

Featureless Void Haskell

The source code for my personal microblog. I've been using it as an excuse to learn Yesod and as a way to do more web work. It includes things like authentication, image uploading with S3, Twitter cross-posting, and generating RSS/JSON feeds for the content.

Personal Website Haskell, HTML, CSS

The source code for my personal website and blog. It's a static website generated by Hakyll and so is largely written in Haskell.
